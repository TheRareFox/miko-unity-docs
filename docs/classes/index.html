Overview of Current Scripts

This is for a general overview of the entire Unity system.

Conceptual Class Diagram 

classDiagram

%% MAIN SYSTEM ENTRY
class BodykunController {
    controls the pose system
    delegates loading and animation
}

class AddFBXController {
    sets up models and dependencies
}

class CameraController {
    handles input and camera movement
}

class ScreenshotController {
    handles pose capture and data export
}

%% LOADERS
class BodykunLoader {
    <<abstract>>
    loads poses
}

class BodykunWebGLLoader
class BodykunSwiftLoader

%% ANIMATION / RIGGING
class AnimationController {
    applies pose animations
}

class HandPosing {
    controls finger muscle movement
}

%% JOINT SYSTEM
class JointLoader {
    initializes joint GameObjects
}

class JointManager {
    tracks and manages joints
}

class SetTarget {
    enables transform gizmos
}

%% STRUCTURE & UTILITIES
class Structure {
    <<static>>
    stores joint mappings
}

class DrawPosekun {
    draws visual skeleton
}

class AdjustModel {
    scales and normalizes model pose
}

class PoseBoundingBox {
    calculates screen space bounds
}

Classes and their public functions

Folder Class Name Description Documentation
../RuntimeGizmo TransformGizmo Draws the Transform Gizmo that we use Poser Documentation: TransformGizmo
. BodykunController Manages the main body model and its interactions. Poser Documentation: BodykunController
./Camera CameraController Handles camera movement, rotation, and zoom. Class: CameraController
./Camera EnableCamera Enables or disables camera functionality. Class: EnableCamera
./Camera GridController Manages the grid visualization in the scene. Class: GridController
./Camera PoseBoundingBox Calculates and manages the bounding box for poses. Class: PoseBoundingBox
./Camera ScreenshotController Captures screenshots of the current scene or model. Class: ScreenshotController
./PoseLoader AddFBXController Adds and configures FBX models, including joints and animations. Poser Documentation: AddFBXController
./PoseLoader BodykunLoader Loads the Bodykun model into the scene.  
./PoseLoader BodykunSwiftLoader Loads Bodykun models with Swift integration.  
./PoseLoader BodyKunWebGLLoader Loads Bodykun models for WebGL platforms.  
PoseLoader/HandPosing HandMuscleGroups Manages muscle groups for hand posing. Class: HandMuscleGroups
PoseLoader/HandPosing HandPosing Handles hand posing and adjustments. Class: HandPosing
PoseLoader/Animations AnimationController Controls animations for the model.  
PoseLoader/Joints JointLoader Loads joint data for the model. Poser Documentation: JointLoader
PoseLoader/Joints JointManager Manages joint interactions and constraints. Poser Documentation: JointManager
PoseLoader/Joints SetTarget Sets target positions for joints. Poser Documentation: SetTarget
Joints/JointRestrictions IKJointRestriction Applies inverse kinematics (IK) restrictions to joints. Class: IKJointRestriction
Joints/JointRestrictions RotationJointRestriction Restricts joint rotation to specified limits. Class: RotationJointRestriction
Joints/JointRestrictions JointRestrictionManager Monitors and controls all Rotation and IK Joint Restrictions Class: JointRestrictionManager
Joints/JointRestrictions JointRestrictionLimits Defines limits for both Rotation and IK Joint Restrictons Class: JointRestrictionLimits
Joints/IKJoints DragJoint Attach to Game Object to be able to drag it Poser Documentation: Drag Joint (IK)
Joints/IKJoints UpdateHintPositions Updates hint positions that will be used for the IK system Poser Documentation: UpdateHintPositions
Joints/DrawJoints DrawLimb Handles updating the setting the color of each bone, drawing the bone and updating its position when joint moves Poser Documentation: DrawLimb
Joints/DrawJoints DrawPosekun Supervises DrawLimb and toggles Body mesh/limb visualisation Poser Documentation: DrawPosekun
./utils AdjustModel Handles scaling and adjustments of body parts. Class: AdjustModel
./utils Colors Provides color utilities for the project. Poser Documentation: Colors
./utils Structure Manages the structure and hierarchy of joints. Poser Documentation: Structure
./utils PoseLibrary Manages a library of poses.  
utils/Indicators SwiftControls Integrates with iOS Swift for joint and pose management. Class: SwiftControls
utils/Indicators StoreOriginalJointForIndicators Stores original joint positions for indicators. Class: StoreOriginalJointForIndicators
utils/Swift NativeStateManager Synchronizes state between Unity and native platforms.  
utils/Swift RootBehavior Manages root-level behaviors for the application.  
utils/UI AutoFillPose Automates loading and management of pose thumbnails and animations.  
utils/UI GenerateCsv Generates CSV files for exporting pose or model data.  
utils/UI PoseButtonController Manage pose selection buttons in the UI.  
utils/UI ResizeWindow Handles resizing UI windows.  
utils/UI SideMenuController Manages the side menu UI interactions.  
UI/HandEditor OpenHandEditor Provides tools for editing hand poses. Class: OpenHandEditor
UI/HandEditor HandWindowDrag Handles drag interactions for hand sliders in the UI. Class: Hand Window Drag
UI/HandEditor HandSliderDrag Manages slider interactions for hand adjustments. Class: HandSliderDrag
UI/HandEditor HandButtonHandler Handles button interactions for hand editing. Class: HandButtonHandler
utils/WebGL RootWebGL Provides WebGL-specific functionality, such as displaying poses. Class: RootWebGL
