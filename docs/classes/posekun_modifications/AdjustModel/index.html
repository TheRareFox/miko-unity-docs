Description

AdjustModel manages dynamic scaling and adjustment of the body model (posekun/bodykun). It modifies limb proportions, height, width, and relative joint positioning based on user input or saved pose data. Used to personalize or match a body pose to specific proportions.

File Location

Assets/Scripts/utils/AdjustModel.cs

Serialized Fields

Name Type Description
rb RigBuilder Reference to the RigBuilder to temporarily disable rig updates during adjustments.
leftArm, rightArm, leftLeg, rightLeg, hips, spine, root, leftHand, rightHand, head, leftFoot, rightFoot, neck Transform References to the key joints and body parts used for scaling and repositioning.

Public Methods

void Initalise()

Links internal joint references to actual GameObjects from the scene using the Structure.jointGameObjects mapping.

void UpdateInitialJointPositionsForScaling()

Stores the initial arm and leg positions used to calculate offsets during scaling.

void ScaleArm(float val)

Scales both upper arms. Disables rigging temporarily for transformation.

void ScaleHands(float val)

Scales both hands.

void ScaleFeet(float val)

Scales both feet.

void ScaleHip(float val)

Horizontally moves legs outward/inward to simulate wider/narrower hips.

void ScaleShoulders(float val)

Horizontally moves arms outward/inward to simulate wider/narrower shoulders.

void ScaleLegs(float val)

Scales both upper legs.

void ScaleTorso(float val)

Scales hips and spine equally. Updates internal joint position references.

void ScaleHeight(float val)

Vertically scales torso (Y or Z depending on camera orientation).

void ScaleThickness(float val)

Horizontally scales hips and spine.

void ScaleHeadWidth(float val)

Scales neck slightly and offsets it to preserve visual integrity.

void ScaleNeck(float val)

Scales the neck down and the head up to simulate a proportional neck.

void ApplyAllScalingToPosekun()

Applies all available scaling adjustments using stored values from RootBehavior.state.

void ResetShoulderHips()

Resets arm and leg joint positions to original locations before scaling started.

Vector3 GetSpinePosition()

Returns the midpoint between left and right arms.

Vector3 GetHipsPosition()

Returns midpoint between legs, slightly adjusted upward.

Vector3 GetUpdatedSpineIndicatorPosition()

Returns average of arm and leg positions to estimate spine indicator center.

Usage Notes

  • Intended for runtime use with UI sliders to let users resize/manipulate the character model.

  • Automatically disables RigBuilder before applying transformations to avoid conflicts.

  • Should be initialized before performing any scale operations.

  • Use in conjunction with BodykunController, Structure, and JointConnectedTo.
