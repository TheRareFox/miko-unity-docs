HandSliderDrag

Description

HandSliderDrag restricts a draggable handle within a triangular UI slider area. It constrains the handle’s movement to remain within (or along the edge of) a triangle defined in UI space and updates a HandPosing component based on the handle’s position. This is used for gesture or hand pose control in a Unity-based interface.

File Location

Assets/Scripts/UI/HandEditor/HandSliderDrag.cs

Serialized Fields

Name Type Description
handle RectTransform Reference to the draggable handle’s RectTransform
sliderAreaObject GameObject Triangle GameObject defining the triangular slider area
handPosing HandPosing Script responsible for applying pose changes to the hand
handButtonHandler HandButtonHandler Script that provides information about the selected finger state

Public Methods

void OnPointerDown(PointerEventData eventData)

Handles initial input when the handle is clicked or touched. Internally calls handleDrag.

void OnDrag(PointerEventData eventData)

Handles drag movement of the pointer and updates the handle’s position accordingly. Internally calls handleDrag.

Private Methods

void Awake()

Initializes the triangle vertices from the attached slider area UI object.

void handleDrag(PointerEventData eventData)

Calculates the local pointer position relative to the triangle. Moves the handle either:

  • to the pointer location (if within triangle), or
  • to the nearest point on the triangle boundary (if outside)

Also updates the hand pose based on the new position.

void ModifyHand(Vector2 pos)

Uses the handle’s position to compute finger open/close and in/out values and applies them to the appropriate hand via the HandPosing component.

bool PointInTriangle(Vector2 p)

Checks if a given point lies inside the triangle defined by sliderArea.a, b, and c.

Vector2 ClosestPointOnTriangle(Vector2 p)

Finds the nearest point on the triangle’s perimeter to a given point.

Vector2 ClosestPointOnLine(Vector2 a, Vector2 b, Vector2 p)

Finds the nearest point on a line segment from point a to b.

Vector3 BarycentricCoords(Vector2 p)

Computes the barycentric coordinates of a point relative to the triangle, which can be used for interpolation or gesture control.

Usage Notes

  • This script is already attached to TriangleSliderUI.
  • The BarycentricCoords are currently not being used, as we actually only need to modify the state of 2 values: Open-Narrow and Open-Wide. The close value automatically defaults when both of them are at 0. However, it may be good to have just in case.
