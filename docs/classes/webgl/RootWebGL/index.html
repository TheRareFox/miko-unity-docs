Description

Handles WebGL-specific behavior and provides a bridge between the Unity WebGL build and external UI-based pose interaction features. Manages visibility of pose elements, scaling adjustments, and indicator display for joints.

File Location

Assets/Scripts/utils/Indicators/StoreOriginalJointForIndicators.cs

Serialized Fields

Field Type Description
bkSkel GameObject The root GameObject representing the skeleton visualization.
bodykunController BodykunController Reference to the Bodykun controller script for managing pose states.
drawPosekun DrawPosekun Reference to the class that draws pose lines.
adjustModel AdjustModel Serialized reference for scaling body parts.

Public Methods

displayOldPose(string display)

Toggles the old pose visualization setup by reassigning drawing connections between joints and triggering re-rendering of poses.

Note: displayOldPose does not work for now, cannot find a way to successfully update the joints

restrictCameraRotation(string restrict)

Enables or disables camera rotation restrictions and resets camera orientation accordingly.

displayBKSkel(string display)

Toggles visibility of the skeletal model (bkSkel).

displayPosekun(string display)

Toggles visibility of the posekun joint visualization model.

scaleHeight(float scaleHeightVal)

Scales the height of the model using AdjustModel.

scaleThickness(float scaleThicknessVal)

Scales model thickness using AdjustModel.

scaleTorso(float scaleTorsoVal)

Scales torso dimensions.

scaleArm(float scaleArmVal)

Scales arm length or thickness.

scaleLegs(float scaleLegsVal)

Scales leg length or thickness.

scaleFeet(float scaleFeetVal)

Scales foot size.

scaleHands(float scaleHandsVal)

Scales hand size.

scaleHeadWidth(float scaleHeadWidthVal)

Scales the width of the head.

scaleNeck(float scaleNeckVal)

Scales the neck dimensions.

scaleHips(float scaleHipsVal)

Scales the hip width or thickness.

scaleShoulders(float scaleShouldersVal)

Scales shoulder width.

Usage Notes

  • This is used to handle communication between Website and WebGL. On the Web, you can call functions of the Root object, which will be handled by this class. Update this class if we need to have extra communications between Root and Web
