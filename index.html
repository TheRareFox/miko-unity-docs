Getting started

If you are already familiar with Unity and installed the project, skip to here:

Requirements

  • Unity Editor + Unity 2022.3.45f1
  • Visual Studio Code (or your preferred C# code editor)

Installation

  1. Clone or download the project: git clone https://github.com/MikoWorld/miko-unity

    a. Note: This repo is private. Please request access from JiYao or other employees for access

  2. Add the project to Unity

    a.In Unity Hub, click Add project

    b. Select the folder: miko-unity/UnityProject

  3. Install required packages

    a.Open the Unity Editor

    b. Go to Window → Package Manager

    c. Install any missing dependencies listed under “In Project”

  1. Setting up recommended automatic file styler in VSCode

    a. Install the C# Extension in VSCode

    b. Add this to your .vscode/settings.json:

{
  "editor.formatOnSave": true,
  "[csharp]": {
    "editor.defaultFormatter": "ms-dotnettools.csharp"
  }
}

Open Your First Scene

Open and Test Run Scene

  1. Open the Unity Project inside Unity

  2. It should look something like this. Check the scene to make sure you have loaded the Main LoadAll Scene. You can change the scene by going into Assets/Scenes and double clicking on LoadAll

  3. Click the play button at the top once the scene is loaded.

  4. Now the scene is loaded and you can move the body!
